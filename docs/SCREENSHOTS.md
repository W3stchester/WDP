# اسکرین‌شات‌ها

برای افزودن اسکرین‌شات‌های واقعی، اپ را اجرا کنید و از بخش‌های مختلف عکس بگیرید:

## نحوه‌ی گرفتن اسکرین‌شات

1. اپ را اجرا کنید:
   ```bash
   npm run dev
   ```

2. مرورگر را روی `http://localhost:5173` باز کنید

3. از بخش‌های مختلف اسکرین‌شات بگیرید:
   - **Windows:** `Win + Shift + S` (Snipping Tool)
   - **macOS:** `Cmd + Shift + 4`
   - **Linux:** `gnome-screenshot -a` یا ابزار مشابه

4. فایل‌ها را با این نام‌ها در این پوشه ذخیره کنید:

## فایل‌های مورد نیاز

| فایل | محتوا |
|------|-------|
| `light-main.png` | صفحه اصلی در حالت روشن |
| `dark-main.png` | صفحه اصلی در حالت تاریک |
| `add-form.png` | مودال افزودن برنامه جدید |
| `jalali-picker.png` | تقویم شمسی |
| `time-picker.png` | تایم‌پیکر سفارشی |
| `custom-reviews.png` | مودال مرورهای اختصاصی |

## نکته

اگر می‌خواهید اسکرین‌شات‌ها را با اندازه‌ی یکنواخت بگیرید:
- از DevTools مرورگر استفاده کنید (`F12`)
- در حالت device toolbar (`Ctrl+Shift+M`) یک دستگاه موبایل انتخاب کنید
- عرض ۴۰۰px مناسب است

پس از افزودن فایل‌ها، تغییرات را commit کنید:
```bash
git add docs/screenshots/
git commit -m "docs: add screenshots"
git push
```
