# Changelog

تمام تغییرات قابل توجه این پروژه در این فایل ثبت می‌شود.

فرمت بر اساس [Keep a Changelog](https://keepachangelog.com/fa/1.0.0/) است.

## [1.4.6] — 2026-01-15

### Added
- سیستم Toast برای اعلان‌های خطا با نوار پیشرفت
- پشتیبانی از arrow keys کیبورد در فیلدهای عددی
- کامپوننت `WheelNumberInput` مشترک (استخراج‌شده از DailyForm)

### Changed
- خطاهای فرم از inline به پاپ‌آپ Toast تغییر کردند
- حداکثر ۳ پاپ‌آپ همزمان (قدیمی‌ترین هنگام اسپم حذف می‌شود)

## [1.4.5] — 2026-01-14

### Added
- نمایش badge مرور روی کارت برنامه‌ها
- پشتیبانی از backdrop-blur در تمام مودال‌ها

### Changed
- بازنویسی حرفه‌ای تمام متن‌ها (دکمه‌ها، خطاها، راهنماها)
- کامنت‌های کد به انگلیسی کوتاه تبدیل شدند
- روشن‌تر کردن بکگراند تم تاریک

### Fixed
- **باگ بحرانی:** `cascadeEditToReviews` برای مرورهای اختصاصی خراب بود (از `REVIEW_SCHEDULE` استفاده می‌کرد به‌جای `reviewBadge` ذخیره‌شده)

### Removed
- فایل `App.css` (unused — قالب Vite پیش‌فرض)
- ۹ تابع unused از `dateUtils.js`
- ۲ تابع unused از `timeUtils.js`

## [1.4.4] — 2026-01-13

### Added
- پشتیبانی از درگ (ماوس + لمس) در تایم‌پیکر
- انیمیشن smooth snap در تایم‌پیکر
- پنجره‌ی تأیید حذف با `ConfirmModal`

### Changed
- حذف منطق دو‌مرحله‌ای «حذف با مرورها؟»
- لیست برنامه‌ها در موبایل تک‌ستونه شد

### Fixed
- outline ناخواسته روی دکمه‌ها (با Space)
- عملکرد مجدد دکمه‌ها با Space/Enter
